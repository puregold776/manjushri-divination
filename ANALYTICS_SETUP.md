# 網站流量分析工具設定指南

## 方法 1: Google Analytics 4 (GA4) - 最推薦

### 優點
- ✅ 完全免費
- ✅ 功能最完整
- ✅ 可看到即時訪客
- ✅ 詳細的訪客行為分析
- ✅ 可追蹤特定事件（如點擊分享按鈕）

### 設定步驟

1. **建立 Google Analytics 帳戶**
   - 前往：https://analytics.google.com/
   - 使用您的 Google 帳號登入
   - 點擊「開始評估」

2. **建立資源**
   - 帳戶名稱：輸入「文殊占卜」或任何您喜歡的名稱
   - 資源名稱：輸入「manjumo.com」
   - 報表時區：選擇「台灣」
   - 貨幣：選擇「新台幣」

3. **設定資料串流**
   - 平台：選擇「網站」
   - 網站網址：輸入 `https://manjumo.com`
   - 串流名稱：輸入「manjumo 主網站」

4. **取得追蹤代碼**
   - 完成後會看到「評估 ID」，格式像：`G-XXXXXXXXXX`
   - 點擊「標記說明」→「手動安裝」
   - 複製整段 JavaScript 追蹤代碼

5. **安裝追蹤代碼**
   - 將追蹤代碼提供給技術人員
   - 代碼需要加入到網站的 `<head>` 區段

### 可查看的數據

- **即時報表**：目前有多少人在線上
- **流量來源**：訪客從哪裡來
  - 自然搜尋（Google、Yahoo 等）
  - 社群媒體（Line、Facebook 等）
  - 直接訪問（輸入網址或書籤）
  - 推薦連結（其他網站的連結）
- **訪客資訊**：
  - 國家/城市
  - 語言
  - 瀏覽器
  - 作業系統
  - 桌面/手機/平板
- **內容分析**：
  - 最多人看的頁面
  - 平均停留時間
  - 跳出率（只看一頁就離開的比例）
- **轉換追蹤**：
  - 可設定目標，例如「點擊分享按鈕」
  - 追蹤特定卦象的查看次數

## 方法 2: Cloudflare Analytics（簡單但功能較少）

如果您的網域使用 Cloudflare DNS，Cloudflare 會自動提供基本分析。

### 可查看的數據
- 總請求數
- 唯一訪客數
- 頻寬使用量
- 威脅和機器人流量

### 查看方式
1. 登入 Cloudflare
2. 選擇 manjumo.com 網域
3. 點擊「分析」標籤

---

## 方法 3: GitHub Pages 內建統計（非常基本）

GitHub 本身不提供詳細的訪客統計，只能看到：
- Repository 的 clone 數量
- 流量圖表（非常基本）

不建議只依賴這個方法。

---

## 方法 4: 簡易訪客計數器

如果只想要一個簡單的訪客計數，可以使用免費服務：
- https://www.cutercounter.com/
- https://www.freecounterstat.com/

但這些功能很陽春，建議還是用 Google Analytics。

## 推薦配置

對於文殊占卜網站，建議：

**主要工具：Google Analytics 4**
- 完整追蹤所有訪客行為
- 了解哪些卦象最受歡迎
- 分析訪客來源和行為

**輔助工具：Google Search Console**
- 監控 Google 搜尋表現
- 了解哪些關鍵字帶來流量
- 追蹤索引狀態

兩者搭配使用可以得到最完整的網站數據！

---

## 隱私考量

Google Analytics 會收集訪客數據，建議：
1. 在網站加入隱私權政策頁面
2. 說明使用 Google Analytics 收集匿名統計數據
3. 提供選擇退出的方式（GA4 有內建選項）

對於宗教/佛教網站，保持透明度很重要。
